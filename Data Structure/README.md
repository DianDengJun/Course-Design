请在运行程序前，解压当前压缩包

源代码所在目录：[./Source-Code/](./Source-Code/)

源代码由三部分组成：
 	1. main.cpp
 	2. sqlist.h
 	3. sqlist.cpp

本程序所使用的编译器是 g++ 8.1.0

要编译代码，请确保您的计算机符合以下条件：
 	1. 已经安装了 g++ 8.1.0 编译器
 	2. 使用 Windows 操作系统

编译代码的命令：“`g++ main.cpp sqlist.cpp -o main`”（不含引号），这将生成可执行文件 `main.exe`

示例输入文件[info.txt](./info.txt)，在当前目录下。您可以在运行程序时，使用此文件导入仓库数据。
